Introduction to Rectifier

Rectifier is converting an alternating current or voltage into a direct current or voltage.

Rectifiers are of two types:

(1) Uncontrolled Rectifier(Diode Rectifier): It uses only diodes and the DC output voltage is fixed.

(2)Controlled Rectifier (Thyristorized Rectifier): It uses SCRs and DC output voltage is variable in amplitude. By controlling the firing angle the output DC voltage (after conversion) can be varied.

Types of Semi Converter:

Half Wave Rectifier: Half wave rectifier converts only half cycle of the input AC voltage to DC output voltage and block other half cycle .By controlling the firing angle the output DC voltage can be varied.

Full Wave Rectifier: Full-wave rectifier converts both half cycle of the input AC voltage to DC output voltage by the use of center tapped transformer. By controlling the firing angle the output DC voltage can be varied.

Semi-Converter: Semiconverter is a type of Bridge rectifier in this configuration two SCR are replaced by two diode and can be connected in either arm of the bridge. By controlling the firing angle the output DC voltage can be varied.

Bridge Rectifier: Bridge rectifier is type of full wave rectifier that converts both half cycle of the input AC voltage to DC output voltage. Bridge rectifier does not need center tapped transformer. By controlling the firing angle the output DC voltage can be varied.

Asymmetrical : In a asymmetrical configuration the cathode of one SCR and a cathode of one diode are at the same potential.

Symmetrical : In a symmetrical configuration the cathode of two SCRs are at the same potential so their gates can be connected and single gate pulse can be used for triggering either SCR.
